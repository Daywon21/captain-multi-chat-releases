## Quick start

1. Install OBS Studio if you have not already.
2. Download and run **CaptainMultiChat-Setup.exe** (close OBS first).
3. In OBS, open **Tools â†’ WebSocket Server Settings** and turn on **Enable WebSocket server** (default port **4455**). This is required for the on-stream chat overlay.
4. Open OBS â†’ **Docks** â†’ **Captain Multi Chat**.
5. Click **Settings**, connect your platforms, then under **Stream overlay (OBS)** choose your chat **Browser source**. Captain Multi Chat wires the overlay URL and per-scene layout for you.

If the dock says it cannot connect to OBS, double-check that WebSocket server is enabled and that OBS is running.
