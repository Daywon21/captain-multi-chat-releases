Captain Multi Chat — User Data Deletion

How to remove Facebook / Meta data used by Captain Multi Chat

Captain Multi Chat stores Facebook connection data only on your computer (encrypted vault + local settings). There is no Captain Multi Chat cloud profile that keeps your Page tokens after you delete local data and revoke the app on Facebook.

Delete from the app (recommended)

  1. Open the Captain Multi Chat dock in OBS
  2. Open Settings
  3. Under Facebook, click Log out

That clears the stored Facebook user token, Page token, and Page selection from the local vault.

Revoke access on Facebook

  1. Visit Facebook → Settings → Apps and websites
  2. Find Captain Multi Chat (or the Meta app name you authorized)
  3. Remove / revoke access

Delete all local Captain Multi Chat data (Windows)

Fully quit OBS, then delete the app data folder:

  • %APPDATA%\CaptainMultiChat

That removes the encrypted vault, config, and related local state for all platforms — not only Facebook.

Request help

If you need confirmation that Facebook authorization was cleared, revoke the app in Facebook settings (above) and open an issue at captain-multi-chat-releases issues.
